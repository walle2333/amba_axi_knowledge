# Reading Plan

## Phase 1: Framework

- 建立目录结构。
- 建立资料登记格式。
- 建立笔记索引和主题列表。

## Phase 2: Primary Sources

- 收集 Arm 官方 AMBA、AXI、ACE、CHI 相关资料。
- 登记规格版本、URL、本地文件和访问日期。
- 标记无法自动下载的资料。

## Phase 3: Core AXI Notes

- AMBA family 总览。
- AXI4 五通道、握手、读写事务。
- burst、response、WSTRB、unaligned transfer、narrow transfer。
- ID、outstanding、ordering。

## Phase 4: SoC Practice

- interconnect、crossbar、NoC。
- AXI-to-APB bridge、CDC、width conversion。
- CPU、DMA、DDR、外设、NPU/GPU/ISP 场景。
- performance、QoS、security、cache coherency。

## Phase 5: Verification and Examples

- RTL 示例。
- timing diagrams。
- assertion 和 verification checklist。
- common bugs 和 debug checklist。

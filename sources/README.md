# Sources

该目录保存 AMBA AXI 相关原始参考资料。原则上原始资料不直接修改，所有来源、URL、访问日期、版本、可靠性和本地路径统一登记到 `metadata/sources.yaml`。

## Categories

- `arm-official/`: Arm 官方规格、官方技术文档、官方培训资料。
- `vendor-docs/`: EDA/IP/FPGA/SoC 厂商公开文档。
- `open-source/`: 开源 AXI IP、验证环境和项目文档。
- `papers/`: 论文、白皮书和技术报告。
- `web-articles/`: 高质量网页文章、课程笔记和博客归档。

## Reliability

```text
primary    Arm 官方资料。
secondary  厂商公开资料。
tertiary   博客、课程、GitHub 文档、论坛讨论。
```
